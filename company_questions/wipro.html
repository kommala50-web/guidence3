Two Sum (Return indices)
Hashmap — O(n)
Solution / Approach: Iterate through the array, storing each number and its index in a hashmap. For each number, check if the complement (target - number) exists in the map. If yes, return the pair of indices. 
def two_sum(nums, target):
    map = {}
    for i, num in enumerate(nums):
        if target - num in map:
            return [map[target - num], i]
        map[num] = i
Time Complexity: O(n) — each element is processed once.
Space Complexity: O(n).
Tip: Clarify if array is sorted before coding.
Check if number is palindrome (without converting to string)
Math — O(log n)
Solution / Approach: Reverse half of the number and compare with first half. 
def is_palindrome(x):
    if x < 0 or (x % 10 == 0 and x != 0):
        return False
    rev = 0
    while x > rev:
        rev = rev * 10 + x % 10
        x //= 10
    return x == rev or x == rev // 10
O(log n) time, O(1) space.
Tip: Avoid converting to string — use math.
Valid Parentheses
Stack — O(n)
Solution: Use stack to match brackets. 
def is_valid(s):
    stack = []
    mapping = {")":"(", "]":"[", "}":"{"}
    for c in s:
        if c in mapping.values():
            stack.append(c)
        elif c in mapping:
            if not stack or stack.pop() != mapping[c]:
                return False
    return not stack
O(n) time, O(n) space.

Remove duplicates from sorted array
Two-pointer — O(n)
Solution: Use two pointers to overwrite duplicates. 
def remove_duplicates(nums):
    if not nums:
        return 0
    i = 0
    for j in range(1, len(nums)):
        if nums[j] != nums[i]:
            i += 1
            nums[i] = nums[j]
    return i + 1
O(n) time, O(1) space.

Climbing Stairs
DP / Fibonacci — O(n)
Solution: Each step = sum of previous two steps. 
def climb_stairs(n):
    if n <= 2:
        return n
    a, b = 1, 2
    for _ in range(3, n + 1):
        a, b = b, a + b
    return b

Binary Search
Divide & Conquer
Solution: Binary search on sorted array. 

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1
O(log n) time, O(1) space.

Detect Cycle in Linked List
Floyd’s Tortoise & Hare
Solution: Slow and fast pointers. 

def has_cycle(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            return True
    return False
O(n) time, O(1) space.

Reverse Linked List
Iterative / Recursive
Solution: Reverse pointers iteratively. 

def reverse_list(head):
    prev = None
    curr = head
    while curr:
        nxt = curr.next
        curr.next = prev
        prev = curr
        curr = nxt
    return prev
O(n) time, O(1) space. Recursive also possible.

Merge Two Sorted Linked Lists
Linked List / Merge
Solution: Use dummy node & merge iteratively. 

def merge_two_lists(l1, l2):
    dummy = cur = ListNode(0)
    while l1 and l2:
        if l1.val < l2.val:
            cur.next = l1
            l1 = l1.next
        else:
            cur.next = l2
            l2 = l2.next
        cur = cur.next
    cur.next = l1 or l2
    return dummy.next
O(n+m) time, O(1) space.

House Robber
Dynamic Programming
Solution: Max loot at i = max(prev, prev2 + nums[i]). 

def rob(nums):
    prev = curr = 0
    for n in nums:
        prev, curr = curr, max(curr, prev + n)
    return curr
O(n) time, O(1) space. Avoid consecutive houses.

Top K Frequent Elements
Heap / Hashmap
Approach: - First, count the frequency of each element using a hashmap (Counter). - Then, use a max-heap (or nlargest in Python) to extract the top K frequent elements efficiently. - This avoids sorting the entire array and focuses only on the K most common items. 

from collections import Counter
import heapq

def topKFrequent(nums, k):
    count = Counter(nums)
    return heapq.nlargest(k, count.keys(), key=count.get)
O(n log k) time, O(n) space — counting + heap operations.

Product of Array Except Self
Prefix / Suffix
Approach: - We can’t use division, so compute products in two passes. - In the first pass, multiply elements from the left (prefix). - In the second pass, multiply elements from the right (suffix). - This ensures every index holds the product of all numbers except itself. 

def product_except_self(nums):
    n = len(nums)
    res = [1]*n
    prefix = 1
    for i in range(n):
        res[i] = prefix
        prefix *= nums[i]
    suffix = 1
    for i in reversed(range(n)):
        res[i] *= suffix
        suffix *= nums[i]
    return res
O(n) time, O(1) extra space. No division used.

Word Ladder (Shortest Transformation)
Breadth-First Search (BFS)
Approach: - Treat each word as a node in a graph, edges exist between words differing by one letter. - Use BFS to ensure the shortest transformation sequence. - Keep track of visited words to avoid cycles. 

from collections import deque

def ladder_length(begin, end, word_list):
    word_set = set(word_list)
    queue = deque([(begin, 1)])
    while queue:
        word, steps = queue.popleft()
        if word == end:
            return steps
        for i in range(len(word)):
            for c in 'abcdefghijklmnopqrstuvwxyz':
                next_word = word[:i] + c + word[i+1:]
                if next_word in word_set:
                    word_set.remove(next_word)
                    queue.append((next_word, steps + 1))
    return 0
O(N * L²) time — N=words, L=length of each word. BFS ensures minimum steps.

Number of Islands
DFS / BFS on Grid
Approach: - Traverse every cell in the grid. - When a land cell (‘1’) is found, increment island count and mark all connected land using DFS/BFS. - Replace visited land cells with ‘0’ to prevent recounting. 

def num_islands(grid):
    if not grid: return 0
    rows, cols = len(grid), len(grid[0])

    def dfs(r, c):
        if r<0 or c<0 or r>=rows or c>=cols or grid[r][c]=='0':
            return
        grid[r][c] = '0'
        dfs(r+1,c); dfs(r-1,c); dfs(r,c+1); dfs(r,c-1)

    count = 0
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == '1':
                dfs(r, c)
                count += 1
    return count
O(M×N) time — each cell visited once. DFS marks connected lands as one island.

Serialize and Deserialize Binary Tree
DFS / Recursion
Approach: - Serialization: perform preorder traversal (root-left-right). Add “#” for null nodes. - Deserialization: read values in the same order to rebuild tree recursively. - Helps in saving and reconstructing tree structure uniquely. 

def serialize(root):
    res = []
    def dfs(node):
        if not node:
            res.append('#')
            return
        res.append(str(node.val))
        dfs(node.left)
        dfs(node.right)
    dfs(root)
    return ' '.join(res)

def deserialize(data):
    vals = iter(data.split())
    def dfs():
        val = next(vals)
        if val == '#': return None
        node = TreeNode(int(val))
        node.left = dfs()
        node.right = dfs()
        return node
    return dfs()
O(n) time & space — each node processed once.

LRU Cache Implementation
Hashmap + Doubly Linked List
Approach: - Use a **Hashmap** for O(1) key lookup. - Use a **Doubly Linked List (DLL)** to maintain the order of most/least recently used elements. - On every `get()` or `put()`, move the accessed node to the front (most recent). - If cache is full, remove the least recently used node from the tail. 

class LRUCache:
    def __init__(self, capacity):
        self.cap = capacity
        self.cache = {}
        self.dll = DoublyLinkedList()

    def get(self, key):
        if key not in self.cache:
            return -1
        node = self.cache[key]
        self.dll.move_to_front(node)
        return node.val

    def put(self, key, val):
        if key in self.cache:
            node = self.cache[key]
            node.val = val
            self.dll.move_to_front(node)
        else:
            if len(self.cache) == self.cap:
                tail = self.dll.remove_tail()
                del self.cache[tail.key]
            node = Node(key, val)
            self.dll.add_front(node)
            self.cache[key] = node
O(1) for both get() and put() using HashMap + DLL.
Explain why DLL is needed instead of just a list.
