1. Introduction to CSS

Description: CSS (Cascading Style Sheets) is used to style HTML elements. It controls colors, fonts, layouts, spacing, animations, and responsiveness. CSS separates content (HTML) from presentation, making websites more maintainable.

<!DOCTYPE html>
<html>
  <head>
    <style>
      body { background-color: lightblue; }
      h1 { color: navy; }
    </style>
  </head>
  <body>
    <h1>Hello CSS</h1>
  </body>
</html>

2. CSS Syntax & Selectors

Description: CSS uses selectors to target HTML elements and apply styles. A CSS rule has a selector and a declaration block. There are element, class, ID, group, descendant, and attribute selectors.

h1 { color: red; }
.intro { font-size: 18px; }
#main { background-color: yellow; }
h1, h2, h3 { margin-bottom: 10px; }
div p { color: green; }

3. Colors & Backgrounds

Description: CSS allows you to change text color, background color, and background images. Colors can be keywords, hex, RGB, RGBA, HSL. Backgrounds can include images, gradients, and control repeat, size, position.

p { color: #333; }
body { background-color: lightyellow; }
div { 
  background-image: url('bg.png');
  background-repeat: no-repeat;
  background-size: cover;
}
h1 { background: linear-gradient(to right, red, yellow); -webkit-background-clip: text; color: transparent; }

4. Fonts & Text

Description: CSS styles text using font-family, size, weight, style, line-height, letter-spacing, text-transform. Web-safe fonts and Google Fonts can be used. Text alignment and decoration enhance readability and design.

p { 
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  font-style: italic;
  line-height: 1.5;
  letter-spacing: 1px;
  text-align: center;
  text-transform: uppercase;
  text-decoration: underline;
}

5. Box Model

Description: Every HTML element is a rectangular box with content, padding, border, and margin. Understanding box-sizing is essential for layouts. border-box includes padding and border in width/height.

div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
  box-sizing: border-box;
}

6. Display & Visibility

Description: CSS display property controls layout type (block, inline, inline-block, none). Visibility hides elements but keeps space.

p { display: block; }
span { display: inline; }
div { display: inline-block; }
.hidden { display: none; }
.invisible { visibility: hidden; }

7. Positioning

Description: CSS positioning (static, relative, absolute, fixed, sticky) determines how elements are placed in the document. top, left, right, bottom move positioned elements.

div { position: relative; top: 10px; left: 20px; }
header { position: fixed; top: 0; width: 100%; }

8. Floating & Clear

Description: float positions elements left or right. clear prevents elements from wrapping around floated elements. Used in old layout techniques.

img { float: left; margin: 10px; }
div::after { content: ""; clear: both; display: table; }

9. Flexbox

Description: Flexbox is a modern layout system for 1D layouts. Flex containers align items horizontally or vertically using flex-direction, justify-content, align-items, flex-wrap, and more.

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  flex-wrap: wrap;
}
.item { flex: 1 1 200px; }

10. Grid

Description: CSS Grid is for 2D layouts with rows and columns. It offers precise control over placement using grid-template-rows, grid-template-columns, grid-gap, and grid-area.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-gap: 20px;
}
.item { background-color: lightblue; }

11. Pseudo-classes & Pseudo-elements

Description: Pseudo-classes (:hover, :focus, :nth-child) select states. Pseudo-elements (::before, ::after, ::first-letter) insert content or style parts of elements.

a:hover { color: red; }
li:nth-child(odd) { background: #eee; }
p::first-letter { font-size: 24px; color: blue; }

12. Transitions

Description: CSS transitions create smooth animations between property changes, specifying duration, timing-function, and delay.

div {
  background-color: blue;
  transition: background-color 0.5s ease;
}
div:hover { background-color: green; }

13. Animations

Description: CSS animations use keyframes to create complex motion. Define @keyframes and use animation property for duration, iteration, and timing.

@keyframes slide {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
div { animation: slide 2s infinite alternate; }

14. Transforms

Description: Transform rotates, scales, skews, and translates elements in 2D or 3D space.

div { transform: rotate(45deg); scale: 1.2; }

15. Filters & Effects

Description: CSS filters adjust images or elements (blur, grayscale, brightness). Shadows and opacity create visual effects.

img { filter: grayscale(50%); }
div { box-shadow: 2px 2px 10px rgba(0,0,0,0.5); }
p { opacity: 0.8; }

16. Units & Measurements

Description: CSS supports px, em, rem, %, vh, vw, fr, and more for flexible and responsive sizing.

div { width: 50%; font-size: 1.2em; margin: 10px 5vw; }

17. Media Queries (Responsive)

Description: Media queries adapt layout for different screen sizes, devices, and orientations using @media rules.

@media (max-width: 768px) {
  .container { flex-direction: column; }
  body { font-size: 14px; }
}

18. CSS Variables

Description: CSS variables (custom properties) allow reusability and easier maintenance. Defined with --name and accessed with var(--name).

:root {
  --main-color: #00b4d8;
}
p { color: var(--main-color); }

19. Gradients

Description: CSS gradients create smooth color transitions. Linear and radial gradients are commonly used for backgrounds and overlays.

div {
  background: linear-gradient(to right, red, yellow);
}
section {
  background: radial-gradient(circle, blue, lightblue);
}

20. Shorthand Properties

Description: Shorthand combines multiple properties into one, like margin, padding, border, font, background, etc.

div { margin: 10px 20px 30px 40px; padding: 5px 10px; border: 2px solid black; }

21. CSS Functions

Description: CSS provides functions like calc(), min(), max(), clamp(), and url() for dynamic values.

div { width: calc(100% - 40px); }
p { font-size: clamp(14px, 2vw, 20px); }

22. Bonus: Accessibility & Best Practices

Description: Use semantic HTML, proper color contrast, readable fonts, responsive layouts, ARIA attributes, and meaningful focus states for accessibility and SEO.

button { outline: 2px solid #00b4d8; }
img { alt: "Description of image"; }
a { focus { outline: 2px dashed blue; } }
