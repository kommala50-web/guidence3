1. Introduction to HTML

Description: HTML is the standard language to create web pages and structure content.

<!DOCTYPE html>\n<html>\n  <head><title>My Page</title></head>\n  <body><h1>Hello World</h1></body>\n</html>

2. HTML Document Structure

Description: All HTML documents have <html>, <head>, <body> elements.

<!DOCTYPE html>\n<html lang="en">\n  <head><title>Structure</title></head>\n  <body><h1>Heading</h1></body>\n</html>

3. Head Elements

Description: <head> contains metadata, title, links, scripts, styles.

<head>\n  <title>Example</title>\n  <link rel="stylesheet" href="style.css">\n</head>

4. Text Formatting Tags

Description: Tags like <h1>-<h6>, <p>, <strong>, <em> format text content.

<h1>Main Heading</h1>\n<p>This is <strong>important</strong> text.</p>

5. Links and Anchors

Description: <a> tags create links to other pages or sections.

<a href="https://example.com">Visit Example</a>

6. Images

Description: <img> embeds images using src and alt attributes.

<img src="logo.png" alt="Logo">

7. Lists

Description: Ordered (<ol>) and unordered (<ul>) lists structure content.

<ul>\n  <li>Item 1</li>\n  <li>Item 2</li>\n</ul>

8. Tables

Description: <table> organizes tabular data with rows and columns.

<table>\n  <tr><th>Name</th><th>Age</th></tr>\n  <tr><td>Alice</td><td>25</td></tr>\n</table>

9. Forms

Description: Forms collect user input using <form>, <input>, <textarea> and buttons.

<form action="/submit">\n  <input type="text" name="name">\n  <button type="submit">Submit</button>\n</form>

10. Input Types

Description: HTML offers multiple input types: text, email, password, number, checkbox, radio, etc.

<input type="email" name="email">\n<input type="checkbox" name="subscribe">

11. Multimedia (Audio & Video)

Description: <audio> and <video> embed media with controls attribute.

<video src="video.mp4" controls></video>\n<audio src="audio.mp3" controls></audio>

12. Semantic Tags

Description: Tags like <header>, <footer>, <article>, <section> improve structure and accessibility.

<header>Header</header>\n<article>Article content</article>

13. Div & Span

Description: <div> groups block elements, <span> for inline content.

<div class="container"><span>Text</span></div>

14. HTML Entities

Description: Special characters can be added using entities like &, <, > etc.

& < > " '

15. Headings & Paragraphs

Description: Headings <h1>-<h6> organize content; <p> defines paragraphs.

<h1>Heading</h1>\n<p>Paragraph text.</p>

16. Block & Inline Elements

Description: Block elements occupy full width; inline elements fit content.

<div>Block</div>\n<span>Inline</span>

17. Iframes

Description: <iframe> embeds other pages within a page.

<iframe src="https://example.com" width="300" height="200"></iframe>

18. Meta Tags

Description: <meta> tags provide page metadata like charset, viewport, description, and keywords.

<meta charset="UTF-8">\n<meta name="viewport" content="width=device-width, initial-scale=1.0">

19. Forms Validation

Description: HTML5 validation attributes like required, pattern, minlength, maxlength.

<input type="text" required pattern="[A-Za-z]+">

20. HTML5 APIs

Description: APIs like Geolocation, LocalStorage, Canvas enhance web functionality.

navigator.geolocation.getCurrentPosition(...);\nlocalStorage.setItem('key','value');

21. HTML Forms Advanced

Description: Input types like date, color, range, email, number improve user input experience.

<input type="date">\n<input type="color">\n<input type="range">

22. Bonus: Accessibility & SEO

Description: Use semantic tags, alt attributes, ARIA labels, proper headings to improve accessibility and SEO.

<img src="logo.png" alt="Logo">\n<header>...</header>
