1. C++ Basics & Setup

Description:
C++ is a high-performance, general-purpose language that builds on C. It is used in systems programming, game development, embedded systems, and high-performance applications. A C++ program consists of headers, the main() function, and statements within curly braces. Input/output is performed using cin and cout from the <iostream> library. Comments can be single-line (//) or multi-line (/* */).

Example Question: Write a program to input your name and age, then display them using cout.


#include <iostream>
using namespace std;

int main() {
    string name;
    int age;
    cout << "Enter your name: ";
    cin >> name;
    cout << "Enter your age: ";
    cin >> age;
    cout << "Hello, " << name << "! You are " << age << " years old." << endl;
    return 0;
}

2. Data Types & Variables

Description:
Data types specify what kind of data a variable can hold. Common types include int, char, float, double, and bool. Variables must be declared before use. Constants declared with const or constexpr cannot be changed. Type casting converts values from one type to another.

Example Question: Write a program demonstrating constants and type casting.


#include <iostream>
using namespace std;

int main() {
    const double PI = 3.14159;
    int radius = 5;
    double area = PI * radius * radius; // implicit casting
    cout << "Area of circle: " << area << endl;

    double num = 7.8;
    int converted = static_cast<int>(num); // explicit cast
    cout << "Converted value: " << converted << endl;

    return 0;
}

3. Operators

Description:
Operators are symbols that perform operations on variables and values. C++ has arithmetic, relational, logical, bitwise, assignment, increment/decrement, ternary, sizeof, and comma operators.

Example Question: Write a program to demonstrate all arithmetic and logical operators.


#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;

    // Arithmetic
    cout << "a + b = " << a + b << endl;
    cout << "a - b = " << a - b << endl;
    cout << "a * b = " << a * b << endl;
    cout << "a / b = " << a / b << endl;
    cout << "a % b = " << a % b << endl;

    // Logical
    cout << "(a > b) && (b > 0) = " << ((a > b) && (b > 0)) << endl;
    cout << "(a == b) || (b > 0) = " << ((a == b) || (b > 0)) << endl;

    return 0;
}

4. Control Flow

Description:
Control flow statements determine the order in which statements execute: if-else, switch, loops, break, continue, goto.

Example Question: Write a program using if-else and loops to print all even numbers from 1 to 20.


#include <iostream>
using namespace std;

int main() {
    cout << "Even numbers from 1 to 20: ";
    for(int i = 1; i <= 20; i++) {
        if(i % 2 == 0) {
            cout << i << " ";
        }
    }
    cout << endl;
    return 0;
}
