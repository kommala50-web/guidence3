1. Introduction to Java

Description: Java is an object-oriented, platform-independent programming language. It runs on JVM and is widely used for web, desktop, and mobile applications.

// HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

2. Java Environment Setup & HelloWorld

Description: Install JDK, set up environment variables, and run your first Java program using command line or IDE.

// Compile: javac HelloWorld.java
// Run: java HelloWorld
System.out.println("Java setup successful!");

3. Data Types & Variables

Description: Java has primitive types: int, char, float, double, boolean. Variables store values in memory with type safety.

int age = 25;
double salary = 50000.5;
char grade = 'A';
boolean isActive = true;
System.out.println(age + " " + salary + " " + grade + " " + isActive);

4. Operators

Description: Java operators: arithmetic (+,-,*,/), relational (>,<,==), logical (&&,||), bitwise (&,|,^), assignment (=,+=), ternary (? :).

int a = 10, b = 5;
System.out.println(a + b); // 15
System.out.println(a > b); // true
System.out.println(a & b); // 0
System.out.println((a > b) ? a : b); // 10

5. Control Statements

Description: Use if, if-else, switch for decisions; for, while, do-while loops for iterations; break & continue to control flow.

for(int i=1;i<=5;i++){
    if(i==3) continue;
    System.out.println(i);
}

6. Functions / Methods

Description: Methods are reusable code blocks. They can return values, take parameters, and help modular programming.

public static int sum(int a, int b){
    return a + b;
}
System.out.println(sum(5,3)); // 8

7. Arrays

Description: Arrays store multiple values of the same type in contiguous memory. Used for lists, tables, and matrices.

int[] nums = {1,2,3,4,5};
for(int n: nums){
    System.out.println(n);
}

8. Strings

Description: Strings store text data. Common methods: length(), charAt(), substring(), concat(), equals(), indexOf().

String str = "Hello";
System.out.println(str.length()); // 5
System.out.println(str.concat(" World")); // Hello World

9. OOP Concepts (Classes & Objects)

Description: Java is object-oriented. Classes define objects with attributes and behaviors. Create instances and use constructors to initialize.

class Person{
    String name;
    int age;
    Person(String n, int a){ name=n; age=a; }
    void info(){ System.out.println(name + " " + age); }
}
Person p = new Person("Alice", 25);
p.info();

10. Inheritance

Description: Classes can inherit from others using extends. Supports single, multilevel, hierarchical inheritance. Use super keyword for parent access.

class Animal{
    void eat(){ System.out.println("Eating"); }
}
class Dog extends Animal{
    void bark(){ System.out.println("Barking"); }
}
Dog d = new Dog();
d.eat();
d.bark();

11. Polymorphism

Description: Java supports compile-time (method overloading) and runtime polymorphism (method overriding with inheritance).

class Calculator{
    int add(int a,int b){ return a+b; }
    double add(double a,double b){ return a+b; } // Overloading
}

12. Encapsulation & Abstraction

Description: Encapsulation hides data using private fields and getters/setters. Abstraction is achieved with abstract classes and interfaces.

class Account{
    private double balance;
    public void setBalance(double b){ balance=b; }
    public double getBalance(){ return balance; }
}

13. Interfaces & Abstract Classes

Description: Abstract classes define incomplete classes with abstract methods. Interfaces define contracts. A class implements interfaces using implements keyword.

interface Drawable{ void draw(); }
class Circle implements Drawable{
    public void draw(){ System.out.println("Drawing Circle"); }
}

14. Packages & Access Modifiers

Description: Packages organize classes and interfaces into namespaces. Access modifiers (public, protected, private, default) control visibility and encapsulation.

// Example of Access Modifiers
package mypackage;

public class Example {
    private int secret = 42;
    public int getSecret() { return secret; }
}

class Test {
    public static void main(String[] args) {
        Example ex = new Example();
        System.out.println("Secret: " + ex.getSecret());
    }
}

15. Inheritance

Description: Inheritance allows one class to acquire properties and methods of another. Supports single, multilevel, hierarchical, and hybrid inheritance (via interfaces).

// Inheritance Example
class Parent {
    void greet() { System.out.println("Hello from Parent"); }
}

class Child extends Parent {
    void greetChild() { System.out.println("Hello from Child"); }
}

public class Test {
    public static void main(String[] args) {
        Child c = new Child();
        c.greet();        // Inherited method
        c.greetChild();   // Child method
    }
}

16. Polymorphism

Description: Polymorphism allows objects or methods to take multiple forms. Includes compile-time (method overloading) and runtime (method overriding) polymorphism.

// Polymorphism Example
class Animal {
    void sound() { System.out.println("Some sound"); }
}

class Dog extends Animal {
    @Override
    void sound() { System.out.println("Bark"); }
}

public class Test {
    public static void main(String[] args) {
        Animal a = new Dog();
        a.sound(); // Runtime polymorphism
    }
}

17. Abstraction & Interfaces

Description: Abstraction hides implementation details. Abstract classes can have both abstract and concrete methods. Interfaces define method contracts without implementation.

// Abstraction Example
abstract class Shape {
    abstract void draw();
}

class Circle extends Shape {
    void draw() { System.out.println("Drawing Circle"); }
}

interface Drawable {
    void paint();
}

class Square implements Drawable {
    public void paint() { System.out.println("Painting Square"); }
}

public class Test {
    public static void main(String[] args) {
        Shape s = new Circle();
        s.draw();

        Drawable d = new Square();
        d.paint();
    }
}

18. Encapsulation

Description: Encapsulation restricts direct access to class members. Use private fields with public getter/setter methods for controlled access.

// Encapsulation Example
class Person {
    private String name;
    public void setName(String name) { this.name = name; }
    public String getName() { return name; }
}

public class Test {
    public static void main(String[] args) {
        Person p = new Person();

    }

19. Constructors & Destructors

Description: Constructors initialize objects when created. Java does not have destructors; garbage collector handles memory cleanup. Can have parameterized, default, and copy-like constructors.

// Constructor Example

public class Test {

    }
}

21. Static Members

Description: Static fields and methods belong to the class, not instances. Shared among all objects.

class Counter {

    Counter() { count++; }

public class Test {

Description: final can be used with variables (constant), methods (cannot override), and classes (cannot inherit).

29. File Handling in Java

Description: Java provides classes like File, FileReader, FileWriter, BufferedReader, BufferedWriter for reading/writing text and binary files.

import java.io.*;

public class Test {
    public static void main(String[] args) throws IOException {
        BufferedWriter writer = new BufferedWriter(new FileWriter("example.txt"));

30. Serialization

Description: Serialization converts objects into a byte stream for storage or transmission. Use Serializable interface and ObjectOutputStream/ObjectInputStream.

import java.io.*;

class Student implements Serializable {
    String name;
    int age;
    Student(String name, int age) { this.name = name; this.age = age; }
}

public class Test {
    public static void main(String[] args) throws Exception {
        Student s = new Student("Alice", 21);
        ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream("student.ser"));
        out.writeObject(s);
        out.close();

        ObjectInputStream in = new ObjectInputStream(new FileInputStream("student.ser"));
        Student s2 = (Student)in.readObject();
        in.close();
        System.out.println(s2.name + " - " + s2.age);
    }
}

31. JDBC (Java Database Connectivity)

Description: JDBC allows Java programs to connect and interact with databases. Use DriverManager, Connection, Statement, ResultSet.

import java.sql.*;

public class Test {
    public static void main(String[] args) throws Exception {
        Connection con = DriverManager.getConnection("jdbc:mysql://localhost:3306/testdb","root","password");
        Statement stmt = con.createStatement();
        ResultSet rs = stmt.executeQuery("SELECT * FROM students");
        while(rs.next()) {
            System.out.println(rs.getInt("id") + " " + rs.getString("name"));
        }
        con.close();
    }
}

32. Reflection API

Description: Reflection allows inspecting and manipulating classes, methods, and fields at runtime.

import java.lang.reflect.*;

class Student {
    private String name = "Alice";
}

public class Test {
    public static void main(String[] args) throws Exception {
        Class<Student> cls = Student.class;
        Field field = cls.getDeclaredField("name");
        field.setAccessible(true);
        Student s = new Student();
        System.out.println(field.get(s)); // prints Alice
    }
}

34. Design Patterns

Description: Design patterns provide reusable solutions to common software design problems. Examples: Singleton, Factory, Observer, Strategy.

// Singleton Pattern Example
class Singleton {
    private static Singleton instance;
    private Singleton() {}
    public static Singleton getInstance() {
        if(instance == null) instance = new Singleton();
        return instance;
    }
}
public class Test {
    public static void main(String[] args) {
        Singleton s1 = Singleton.getInstance();
        Singleton s2 = Singleton.getInstance();
        System.out.println(s1 == s2); // true
    }
}

41. Stream API Advanced

Description: Stream API allows functional-style operations on collections: filter, map, reduce, collect. Supports parallel processing for performance.

import java.util.*;
import java.util.stream.*;

public class Main {
    public static void main(String[] args) {
        List nums = Arrays.asList(1,2,3,4,5);
        int sum = nums.stream().filter(n->n%2==0).mapToInt(n->n*2).sum();
        System.out.println("Sum of doubled evens: " + sum);
    }
}

46. Mini Projects / Real-world Applications

Description: Apply Java concepts in projects to consolidate learning: Calculator, Banking System, Student Management, Chat Applications, etc.

import java.util.Scanner;

public class Calculator {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Enter 1st number: ");
        double a = sc.nextDouble();
        System.out.print("Enter 2nd number: ");
        double b = sc.nextDouble();
        System.out.print("Enter operator (+,-,*,/): ");
        char op = sc.next().charAt(0);

        double result = 0;
        switch(op){
            case '+': result = a+b; break;
            case '-': result = a-b; break;
            case '*': result = a*b; break;
            case '/': result = a/b; break;
            default: System.out.println("Invalid operator");
        }
        System.out.println("Result: "+result);
    }
}
