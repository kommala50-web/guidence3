1. Introduction to DBMS

Description: Database Management Systems (DBMS) are software systems designed to store, manage, and retrieve data efficiently. They provide mechanisms for data integrity, security, concurrency, and backup. Types include relational (RDBMS), NoSQL, hierarchical, and network databases.

// Example: Creating a table in SQL
CREATE TABLE Students (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    age INT
);

1. Introduction to CSS

Description: CSS (Cascading Style Sheets) is used to style HTML elements. It controls colors, fonts, layouts, spacing, animations, and responsiveness. CSS separates content (HTML) from presentation, making websites more maintainable.

<!DOCTYPE html>
<html>
  <head>

      body { background-color: lightblue; }
      h1 { color: navy; }
    </style>
  </head>
  <body>
    <h1>Hello CSS</h1>
  </body>
</html>
