1. Introduction to Python

Description: Python is a high-level, interpreted programming language known for readability and a large ecosystem. Widely used for web development, data science, automation, scripting, and more.

print("Hello, Python!")  # run with: python file.py

2. Installing Python & Tools

Description: Install from python.org or use package managers. Use a code editor (VS Code), virtual environments (venv), and pip for packages.

# create virtual env
python -m venv venv
# activate (Windows)
venv\Scripts\activate
# install packages
pip install requests numpy pandas

3. Syntax & Variables

Description: Python uses indentation to define blocks. Variables are dynamically typed. Naming conventions: snake_case for variables and functions.

name = "Ramesh"
age = 20
is_active = True
print(f"{name} is {age} years old")

4. Data Types

Description: Core types: int, float, str, bool, list, tuple, set, dict, and None. Use type() to inspect.

x = 10         # int
y = 3.14        # float
s = "text"      # str
lst = [1,2,3]   # list
tpl = (1,2)     # tuple
st = {1,2}      # set
d = {"a":1}     # dict
print(type(x), type(s))

4.1. Input & Output

Description: Take user input using input() and display output with print(). Convert input to numbers with int() or float().

name = input("Enter your name: ")
age = int(input("Enter your age: "))
print(f"Hello {name}, you are {age} years old.")

5. Operators

Description: Arithmetic (+, -, *, /, //, %), comparison (==, !=, >, <), logical (and, or, not), assignment (+=, -=) and membership (in).

a = 7
b = 3
print(a + b, a // b, a % b)
print(3 in [1,2,3])  # True

6. Control Flow

Description: Use if, elif, and else to control flow. Python uses indentation; no braces.

x = 10
if x > 10:
    print("Greater")
elif x == 10:
    print("Equal")
else:
    print("Smaller")

6.1. Type Conversion

Description: Convert between types explicitly using built-in functions. Useful when taking input or performing numeric operations.

x = "100"
print(int(x) + 50)   # 150
print(float("3.14"))
print(str(25) + " apples")

7. Loops

Description: Iterate with for (over iterables) and while. Use break and continue to control loop execution.

for i in range(5):
    print(i)

i = 0
while i < 5:
    print(i)
    i += 1

7.1. Nested Conditions

Description: You can nest if statements to test multiple layers of conditions.

x = 10
if x > 0:
    if x % 2 == 0:
        print("Positive even")
    else:
        print("Positive odd")
else:
    print("Non-positive")

7.2. Loop Control Statements

Description: Use break to exit a loop early, continue to skip to next iteration, and else with loops for no-break completion.

for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Completed without break")

8. Functions

Description: Define reusable logic with def. Support default args, keyword args, *args, **kwargs, and docstrings.

def greet(name="User"):
    """Return greeting string."""
    return f"Hello, {name}"

print(greet("Ramesh"))
print(greet())

9. Scope & Lifetime

Description: Python scopes: local, enclosing, global, and built-in (LEGB). Use global or nonlocal carefully.

x = "global"
def outer():
    x = "enclosing"
    def inner():
        nonlocal x
        x = "modified"
    inner()
    print(x)  # modified
outer()
print(x)  # global

Description: Break code into modules (.py files) and packages (folders with _init_.py). Import using import or from ... import.

12. Data Structures

Description: Lists are mutable ordered collections, tuples are immutable, sets are unordered unique items, dicts are key-value mappings (ordered insertion since Python 3.7).

13. List Comprehensions

Description: Compact way to build lists. Generator expressions create lazy iterators (useful for memory efficiency).

squares = [x*x for x in range(10)]
evens = (x for x in range(100) if x%2==0)  # generator
print(squares[:5], next(evens))

14. Strings & Formatting

Description: Strings are immutable. Use f-strings (Python 3.6+) for readable interpolation, plus common methods like split, join, strip, replace.

17. Classes & Objects (OOP)

Description: OOP in Python models real-world entities. Classes define structure (attributes) and behavior (methods). Use _init for construction, self to access instance data, and repr/str_ for readable output.

class Person:
    def _init_(self, name, age):
        self.name = name   # instance attribute
        self.age = age

    def greet(self):
        return f"Hi, I'm {self.name} and I'm {self.age}"

    def _repr_(self):
        return f"Person({self.name!r}, {self.age})"

p = Person("Ramesh", 20)
print(p.greet())
print(p)  # Person('Ramesh', 20)

18. Inheritance & Polymorphism

Description: Extend classes with inheritance (class Sub(Parent):). Use super() to call parent constructors/methods. Polymorphism: same method name, different implementations.

class Animal:
    def speak(self): return "makes sound"

class Dog(Animal):
    def speak(self): return "barks"

a = Animal()
d = Dog()
for obj in (a, d):
    print(obj.speak())  # demonstrates polymorphism

19. Decorators

Description: Decorators wrap functions or methods to modify behavior (logging, caching, authorization). Can be functions or classes returning callable.

def timer(fn):
    import time
    def wrapper(*args, **kwargs):
        start = time.time()
        result = fn(*args, **kwargs)
        print("Elapsed:", time.time()-start)
        return result
    return wrapper

@timer
def work(n):
    sum(i*i for i in range(n))

work(100000)

20. Generators & Iterators

Description: Generators produce values lazily using yield. Iterators implement _iter and next_. Great for streaming data and memory efficiency.

def count_up(n):
    i = 0
    while i < n:
        yield i
        i += 1

for v in count_up(3):
    print(v)

21. Context Managers

Description: Use with to manage resources. Implement context managers via _enter/exit_ or contextlib.contextmanager.

from contextlib import contextmanager

@contextmanager
def open_read(path):
    f = open(path, 'r')
    try:
        yield f
    finally:
        f.close()

with open_read('notes.txt') as f:
    print(f.readline())

22. Lambda & Functional Tools

Description: Small anonymous functions with lambda. Combine with map, filter, and functools.reduce for functional-style processing.

from functools import reduce
nums = [1,2,3,4]
print(list(map(lambda x: x*2, nums)))
print(list(filter(lambda x: x%2==0, nums)))
print(reduce(lambda a,b: a+b, nums))

27. Asyncio & Concurrency

Description: Write asynchronous code with async/await using asyncio. For CPU-bound tasks use multi-processing; for I/O-bound use asyncio or threads.

import asyncio

async def say(msg, delay):
    await asyncio.sleep(delay)
    print(msg)

async def main():
    await asyncio.gather(say("Hi",1), say("Hello",0.5))

asyncio.run(main())

32. Web Development (Flask Intro)

Description: Flask is a lightweight web framework. Create routes, render templates, and handle requests easily. For larger apps use blueprints and WSGI servers.

from flask import Flask, jsonify, request
app = Flask(_name_)

@app.route("/")
def home():
    return "Hello Flask"

@app.route("/api", methods=["POST"])
def api():
    data = request.json
    return jsonify({"received": data})

if _name_ == "_main_":
    app.run(debug=True)
